# Инструкция по установке сайта

## Описание

Сайт строительной компании Максима Терехичева реализован на чистом HTML + CSS с формой обратной связи, отправляющей данные через PHP скрипт.

## Структура файлов

```
public/
├── index.html       # Главная страница сайта
├── styles.css       # Все стили сайта
├── send.php         # PHP скрипт для отправки формы на почту
├── favicon.ico      # Иконка сайта
└── src/
    └── assets/      # Папка с изображениями
```

## Установка на хостинг

### Шаг 1: Загрузка файлов

Загрузите все файлы из папки `public/` на ваш хостинг:
- `index.html`
- `styles.css`
- `send.php`
- папку `src/` со всеми изображениями
- `favicon.ico`

### Шаг 2: Настройка PHP скрипта

Откройте файл `send.php` и измените настройки почты:

```php
$to = 'info@sk-terekhichev.ru'; // ЗАМЕНИТЕ на ваш email
```

### Шаг 3: Проверка работы

1. Откройте сайт в браузере
2. Прокрутите до формы обратной связи
3. Заполните форму и отправьте заявку
4. Проверьте, что письмо пришло на указанный email

## Требования к хостингу

- PHP 7.0 или выше
- Функция `mail()` должна быть включена
- Поддержка отправки писем с сервера

## Настройка почты

Если письма не приходят, проверьте:

1. **Включена ли функция mail()** - свяжитесь с хостингом
2. **Настройки SPF записей** - добавьте домен в SPF записи
3. **Проверьте папку "Спам"** - письма могут попадать туда

### Альтернативная настройка через SMTP

Если `mail()` не работает, можно использовать SMTP. Для этого:

1. Установите библиотеку PHPMailer
2. Измените `send.php` для использования SMTP
3. Укажите настройки SMTP вашего почтового сервера

Пример настройки SMTP (требует PHPMailer):

```php
<?php
require 'PHPMailer/PHPMailerAutoload.php';

$mail = new PHPMailer;
$mail->isSMTP();
$mail->Host = 'smtp.example.com';
$mail->SMTPAuth = true;
$mail->Username = 'your-email@example.com';
$mail->Password = 'your-password';
$mail->SMTPSecure = 'tls';
$mail->Port = 587;

$mail->setFrom('noreply@example.com', 'Website');
$mail->addAddress('info@sk-terekhichev.ru');
$mail->Subject = 'Новая заявка';
$mail->Body = $email_message;

if(!$mail->send()) {
    echo json_encode(['success' => false, 'message' => 'Ошибка отправки']);
} else {
    echo json_encode(['success' => true, 'message' => 'Отправлено']);
}
?>
```

## Редактирование контента

### Изменение текста

Откройте `index.html` и найдите нужный раздел. Текст находится внутри HTML тегов.

### Изменение изображений

Замените файлы в папке `src/assets/` на свои изображения с теми же названиями.

### Изменение цветов

Откройте `styles.css` и измените переменные в разделе `:root`:

```css
:root {
    --accent: hsl(12, 52%, 55%);  /* Основной акцентный цвет */
    --primary: hsl(0, 0%, 42%);   /* Основной серый цвет */
    /* и другие... */
}
```

### Изменение контактов

В файле `index.html` найдите секцию с `id="contacts"` и измените:
- Номер телефона
- Email
- Адрес

Также измените эти данные в шапке сайта (раздел `<header>`).

## Техническая поддержка

При возникновении проблем:
1. Проверьте консоль браузера на наличие ошибок (F12)
2. Проверьте логи ошибок PHP на хостинге
3. Убедитесь, что все файлы загружены корректно

## Безопасность

Скрипт `send.php` включает базовую защиту от спама:
- Проверка метода запроса (только POST)
- Валидация данных
- Защита от XSS атак
- Фильтрация HTML тегов

Рекомендуется дополнительно:
- Добавить Google reCAPTCHA
- Настроить rate limiting (ограничение частоты запросов)
- Использовать HTTPS

## Оптимизация

Для улучшения скорости загрузки:
1. Сжимайте изображения (используйте TinyPNG или аналоги)
2. Включите gzip сжатие на сервере
3. Добавьте кеширование статических файлов
4. Используйте CDN для изображений
