# СК Максима Терехичева - Сайт

Сайт строительной компании. HTML + CSS + PHP.

## Структура

```
public/
├── index.html       # Главная страница
├── styles.css       # Стили
├── send.php         # Обработка формы
└── src/assets/      # Изображения
```

## Установка

1. Скопируйте все файлы из `public/` на хостинг
2. Укажите свой email в `send.php`
3. Готово!

## Требования

- PHP 7.0+
- Функция mail() включена

## Редактирование

- **Текст**: `index.html`
- **Стили**: `styles.css`
- **Email**: `send.php`
- **Изображения**: `src/assets/`
