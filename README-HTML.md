# Сайт СК Максима Терехичева (HTML версия)

Чистый HTML + CSS сайт с PHP формой обратной связи.

## Быстрый старт

1. Скопируйте все файлы из папки `public/` на ваш хостинг
2. Откройте файл `send.php` и укажите ваш email
3. Готово! Сайт работает.

## Структура

```
public/
├── index.html       - Главная страница
├── styles.css       - Все стили
├── send.php         - Отправка формы на почту
├── .htaccess        - Настройки Apache (опционально)
└── src/assets/      - Изображения
```

## Редактирование

### Контент
- Откройте `index.html` в любом текстовом редакторе
- Найдите нужный раздел и измените текст

### Стили
- Откройте `styles.css`
- Измените переменные в начале файла (цвета, шрифты)

### Email для заявок
- Откройте `send.php`
- Измените строку: `$to = 'ваш-email@example.com';`

## Требования

- PHP 7.0+
- Функция mail() или SMTP
